摘要

本规范定义了层叠样式表第 2 级。CSS 是一种样式表语言，允许作者和用户为结构化文档（例如 HTML 文档和 XML 应用程序）附加样式（例如字体和间距）。通过将文档的呈现样式与文档内容分离，CSS 简化了网页创作和网站维护。

CSS 2.2 是 CSS 第 2 级的第二次修订版 [CSS2]。它修正了 CSS 2.1（CSS 第 2 级的第一次修订版）中的一些错误。它不是最新版本的 CSS。有关构成整个 CSS 的规范概述，请参阅“CSS 快照”[CSS]。

CSS 第 2 级支持特定媒体的样式表，以便作者可以根据视觉浏览器、听觉设备、打印机、盲文设备、手持设备等调整其文档的呈现方式。它还支持内容定位、表格布局、国际化功能以及与用户界面相关的一些属性。
