附录 D. HTML 4 默认样式表

本附录是信息性的，而不是规范性的。

本样式表描述了基于对当前 UA 实践的广泛研究的所有 HTML 4 ([HTML4]) 元素的典型格式化。鼓励开发者在他们的实现中使用它作为默认样式表。

某些 HTML 元素的完整呈现无法在 CSS 2.2 中表达，包括 替换 元素（"img"、"object"）、脚本元素（"script"、"applet"）、表单 控件元素和框架元素。

对于其他元素，遗留的呈现可以用 CSS 描述，但解决方案是移除该元素。例如，可以通过将 CSS 声明附加到其他元素（例如， DIV）来替换 FONT 元素。同样，演示属性的遗留呈现（例如，TABLE 上的 "border" 属性）可以用 CSS 描述，但源文档中的标记必须更改。
