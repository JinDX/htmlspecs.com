摘要

本规范定义了可缩放矢量图形（SVG）版本 2 的特性和语法。SVG 是一种基于 XML 的语言，用于描述二维矢量和混合矢量/光栅图形。SVG 内容可样式化，能够缩放到不同的显示分辨率，可以独立查看、与 HTML 内容混合或通过在其他 XML 语言中使用 XML 命名空间嵌入。SVG 还支持动态变化；可以使用脚本创建交互文档，并可以使用声明性动画特性或脚本进行动画处理。
