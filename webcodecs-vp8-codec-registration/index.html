摘要

此注册已录入 [webcodecs-codec-registry]。 它描述了 VP8 的：(1) 完全限定的编解码器字符串， (2) 编解码器特有的 EncodedVideoChunk [[internal data]] 字节，(3) VideoDecoderConfig.description 字节， 以及 (4) EncodedVideoChunk [[type]] 的值。

此注册无意包含任何关于某种 编解码器格式是否受知识产权主张约束的信息。如果实现者和 作者打算实现或使用特定的编解码器格式，建议他们在此事项上寻求 适当的法律顾问。WebCodecs 的实现者不被要求支持 VP8 编解码器。

此注册是非规范性的。

1. 完全限定的编解码器字符串

编解码器字符串为 "vp8"。

2. EncodedVideoChunk 数据

EncodedVideoChunk 的 [[internal data]] 预期为 一个帧，如 [VP8] 第 4 节和附录 A 中所述。

3. VideoDecoderConfig description

description 不用于此编解码器。

4. EncodedVideoChunk type

如果某个 EncodedVideoChunk 的 [[type]] 为 key， 则该 EncodedVideoChunk 预期 包含一个 key_frame 为 true 的帧，如 [VP8] 第 19.1 节所定义。

5. 隐私考量

请参阅 隐私 考量一节，该节位于 [WEBCODECS] 中。

6. 安全考量

请参阅 安全 考量一节，该节位于 [WEBCODECS] 中。
